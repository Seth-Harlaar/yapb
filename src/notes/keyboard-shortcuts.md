Muscle memory is underrated. Every time I reach for the mouse to do something I've done a hundred times, I'm paying a small tax. Learning the keyboard shortcut the first time costs maybe thirty seconds. Not learning it costs thirty seconds every single time after that, compounded forever. The math is obvious. The hard part is pausing long enough in the moment to actually look it up instead of just reaching for the mouse again.
